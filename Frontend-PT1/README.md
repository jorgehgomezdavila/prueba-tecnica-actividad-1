# Frontend-PT1 (Angular + Tailwind)

Proyecto base generado **como si** se hubiera creado con:

```bash
ng new Frontend-PT1 --style=css --routing --ssr=false
```

y luego se le agregó TailwindCSS.

## Requisitos
- Node.js 18+ recomendado
- Angular CLI (opcional, pero recomendado)

## Instalación
```bash
npm install
```

## Levantar en desarrollo
```bash
npm start
```

## Tailwind
Tailwind está habilitado vía `postcss.config.js` y el archivo `src/styles.css`.
